# node-hint

A command line interface for jshint.

## Install

    npm install node-hint

## Using

    node-hint path path2 [options]

## Options

    --config path/to/config.json

## Running Tests

    npm install sinon nodeunit

## Jake Commands

    // run tests
    jake test

    // run jshint
    jake lint
